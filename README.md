# ProTimer.club

**ProTimer.club** is a modern solution for athletes, providing precise time measurement, result recording, and analysis to enhance training efficiency. Our devices are suitable for various sports and competitions, offering convenience, accuracy, and cloud integration to users.

<img src="https://github.com/renat2985/protimer.club/blob/main/doc/protimer3.png" height="250px" alt="ProTimer Image 2"><img src="https://github.com/renat2985/protimer.club/blob/main/doc/protimer2.png" height="250px" alt="ProTimer Image 1">

## Key Features

- **Precise time measurement:** Accuracy up to 0.003 seconds.
- **Ease of use:** Simple system setup accessible to everyone.
- **Cloud integration:** Export data to ProTimer.club for storing and analyzing training history.
- **Social network:** Share achievements and follow the results of friends.
- **Versatility:** Support for various modes and features for team and individual training.

## System Components

### Master Point
The main module of the system that ensures control and accurate time measurement.  
**Features:**
- Wi-Fi hotspot for system connectivity.
- Chronometer with a built-in display.
- Supports BMX Random Gate Start, result history, and cloud export.
- Connection to electromagnetic barriers.

### Points (Modules)

Additional modules to expand the system's capabilities:  
- **Basic:** Essential features for individual use.  
- **PRO:** Equipped with a display, channel selection, and additional modes.

#### Start, Interim, and Finish Points

Modules for starting, intermediate splits, and stopping measurements:  
- Support up to 15 intermediate points.  
- Operate up to 250 meters from the Master Point.  

#### PRO Point

An advanced module with a display for professional use.  
**Modes:**
- Start
- Intermediate split
- Finish
- BMX Random Gate Start
